# IncidentIQ AI

An AI-powered Healthcare Device Operations Platform designed to improve hospital operations through predictive maintenance, intelligent incident management, cybersecurity visibility, and executive dashboards.

## Overview

IncidentIQ AI simulates a modern healthcare technology platform for:

- Medical device inventory
- Predictive maintenance
- AI-generated incident summaries
- Cybersecurity monitoring
- Hospital digital twin visualization
- Executive operational dashboards

This project is intended as a portfolio demonstration using synthetic data. It does not connect to production hospital systems.
